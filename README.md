# Johnny Zhou - Personal Portfolio

[![Live Demo](https://img.shields.io/badge/Live_Demo-Click_Here-blueviolet?style=for-the-badge)](https://johnnyzhou666.github.io/Portfolio/)

A responsive and interactive personal portfolio website built to showcase my software engineering projects, technical skills, and professional background.

## ✨ Features

* **Modern UI/UX:** Clean, responsive design built with React and Bootstrap.
* **Smooth Navigation:** Custom active-state highlighting and smooth scrolling implemented with `react-scroll` and `IntersectionObserver`.
* **Dynamic Animations:** Features custom CSS3 animations, including typewriter effects and floating elements.
* **Mobile Ready:** Fully responsive grid layouts adapted for all screen sizes and devices.

## 🛠 Built With

* **Frontend Framework:** [React.js](https://reactjs.org/)
* **Styling & UI:** [React-Bootstrap](https://react-bootstrap.github.io/) / Custom CSS
* **Scroll Interactions:** [React-Scroll](https://www.npmjs.com/package/react-scroll)
* **Icons:** FontAwesome & Bootstrap Icons
* **Deployment:** GitHub Pages
